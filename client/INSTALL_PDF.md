# PDF Export Setup

Run this command in the client directory:

```powershell
cd "C:\Users\jstickney\Documents\VISUAL STUDIO PROJECTS\TIME TRACKING\client"
npm install jspdf jspdf-autotable
```

This will install:
- `jspdf` - PDF generation library
- `jspdf-autotable` - Plugin for creating tables in PDFs

After installation, the PDF export feature will work automatically.
